# Loan Default Risk Analysis using Statistics & Probability

## Project Overview

This project analyzes loan application data using statistical and probability concepts to evaluate loan default risk. The dataset contains customer information such as age, income, loan amount, credit score, loan term, and default status.

The objective is to understand customer behavior, identify risk factors, and apply statistical techniques for decision-making.

---

## Dataset Information

Dataset Name: loan_applications.csv

Columns:

* Customer_ID
* Age
* Income
* Loan_Amount
* Credit_Score
* Loan_Term
* Default_Status

Total Records: Approximately 5000

---

# Theory Concepts

## Mean

Mean is the average value of a dataset.

Formula:

Mean = Sum of Values / Number of Values

---

## Median

Median is the middle value after arranging data in ascending order.

---

## Mode

Mode is the most frequently occurring value in a dataset.

---

## Standard Deviation

Standard deviation measures how far data values are spread from the mean.

---

## Variance

Variance is the square of standard deviation.

---

## Random Variable

A random variable is a variable whose value depends on the outcome of a random experiment.

Example:
Loan default status (Yes/No)

---

## Conditional Probability

Conditional probability measures the probability of an event occurring given another event has already occurred.

Example:

P(Default | Credit Score < 600)

---

## Bayes Theorem

Bayes Theorem updates probabilities using new information or evidence.

Applications:
Banks use Bayes Theorem to estimate loan default risk.

---

## Empirical Probability

Probability calculated from observed data.

Example:

Number of defaults / Total customers

---

## Theoretical Probability

Probability calculated mathematically before conducting experiments.

---

## Poisson Distribution

Poisson distribution is used for count-based events occurring in a fixed interval.

Example:
Number of loan applications received per day.

---

## Eigenvalues and Eigenvectors

Eigenvalues and Eigenvectors help identify important directions and patterns in data.

Used in:

* PCA
* Machine Learning
* Data Reduction

# Results and Findings

1. Customers with lower credit scores showed higher chances of default.

2. Income distribution was positively skewed.

3. Loan amounts varied significantly among customers.

4. Q-Q Plot indicated data was not perfectly normally distributed.

5. Customers with higher credit scores had lower default probabilities.
