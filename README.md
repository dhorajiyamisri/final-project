# Loan Default Risk Analysis Using Statistics and Probability

# Project Overview

This project analyzes loan default risk using statistical methods, probability concepts, and linear algebra techniques. A synthetic dataset containing 5000 loan applications is used to perform descriptive statistics, probability analysis, data visualization, and vector operations.

# Objectives

- Calculate Mean, Median, and Mode of customer income.
- Compute Range, Variance, and Standard Deviation of loan amounts.
- Estimate the probability of loan default.
- Analyze conditional probability based on credit scores.
- Visualize data using Histograms and Q-Q Plots.
- Apply Linear Algebra concepts such as Dot Product, Norm, and Angle Between Vectors.
- Generate insights regarding customer loan behavior.

# Key Findings

1. Customers with lower credit scores have a higher probability of loan default.
2. Credit score is a significant indicator of repayment behavior.
3. Loan amounts vary considerably among customers.
4. Income distribution is approximately normal.
5. Higher credit scores are associated with lower default risk.

## Conclusion

The project demonstrates how statistics, probability, and linear algebra can be used to analyze loan default risk. The results help identify factors that influence customer repayment behavior and support data-driven decision-making in financial institutions.
